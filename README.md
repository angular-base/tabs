# tabs


## Usage it in your code
    
    <base-tabs direction="horizontal">
      <base-tab title="Tab 1">
        <h1>One</h1>
        <p>Check me out!</p>
      </base-tab>
      <base-tab title="Tab 2">
        <h1>Two</h1>
        <p>I'm a Tab panel with text content!</p>
      </base-tab>
    </base-tabs>

### Attributes
* **`direction: string`** - Should be `horizontal` or `vertical`.
